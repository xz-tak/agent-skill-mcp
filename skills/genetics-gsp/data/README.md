# GSP Data Files

Place your `*_GSP.xlsx` files in this directory.

## Expected File Format

Each file should be named `{TARGET}_GSP.xlsx` where `{TARGET}` is the gene/protein name.

## Required Sheets

| Sheet | Used By | Description |
|-------|---------|-------------|
| **GWAS Summary** | Both | Main GWAS evidence |
| **Study Info** | Biological | Study metadata |
| **OMIM** | Safety | Mendelian phenotypes |
| **OT Gene Burden** | Safety | Rare variant associations |
