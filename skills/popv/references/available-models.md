## Model Selection Guide

| Scenario | Recommended Model |
|----------|------------------|
| Human tissue-specific annotation | `popV_tabula_sapiens_{Tissue}` |
| Human broad annotation (unknown tissue) | `popV_tabula_sapiens_All_Cells` |
| Human compartment-specific | `popV_tabula_sapiens_{Compartment}` (Immune, Stromal, etc.) |
| Mouse 10x data | `popV_tabula_muris_{Tissue}_10x` |
| Mouse Smart-seq2 data | `popV_tabula_muris_{Tissue}_Smart-seq2` |
| Mouse broad annotation | `popV_tabula_muris_All` |
| Mouse data with human reference (cross-species) | Use a `tabula-sapiens` model â€” homolog mapping is automatic |
| Custom organism/tissue not available | Create a custom model (Workflow 2) |

## Model Contents

Each model directory contains:

| File | Description |
|------|-------------|
| `minified_ref_adata.h5ad` | Reference anndata (HuggingFace models) |
| `{MODEL}.h5ad` | Reference anndata (custom models) |
| `scvi/` | scVI latent space model |
| `scanvi/` | ScanVI semi-supervised model |
| `celltypist.pkl` | CellTypist classifier |
| `xgboost_classifier.model` | XGBoost classifier |
| `svm_classifier.joblib` | SVM classifier |
| `harmony_knn_classifier.joblib` | Harmony KNN classifier |
| `scvi_knn_classifier.joblib` | scVI KNN classifier |
| `pynndescent_index.joblib` | BBKNN index |
| `OnClass.*` | OnClass model files |
| `metadata.json` | Model metadata |
| `accuracies.json` | Training accuracies |
| `model.txt` | Human-readable summary |
